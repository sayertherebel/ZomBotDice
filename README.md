# ZomBotDice
Bot Framework v4 implementation of the Zombie Dice game. Supports multiplayer through shared game code.

## Prerequisites

- [.NET Core SDK](https://dotnet.microsoft.com/download) version 2.1
- [AdaptiveCards 2.0rc]
- [LUIS]
